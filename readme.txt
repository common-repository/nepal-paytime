== Description ==
PayTime plugin enables any user with a wordpress site to integrate PayTime payment gateway in their website/web application. This is extension to WooCommerce Plugin.


=== Features === 

Adds PayTime payment gateway for WooCommerce
Translation Ready
Lifetime Free support
Constant updates

== Installation ==
1. Unzip paytime.zip
2. Upload all the files to the /wp-content/plugins/paytime
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin will add PayTime payment gateway for WooCommerce

= Which currency is accepted by the plugin =
ONLY  Nepalese rupee(Rs)

= How can I use plugin? =
Just install and activate this plugin and payment option will be added on WooCommerce.
